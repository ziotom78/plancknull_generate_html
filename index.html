Source code for plancknull_generate_html

This site provides the full documentation of the program plancknull_generate_html, used to produce reports of the null tests produced for the Planck/LFI instrument.

The program has been written using Chicken Scheme. Select one of the files below to read the commented source code:

  • file-utils.scm: functions that deal with files (e.g. producing a GIF file from a Healpix map in FITS format).
  • generator.scm: the main program
  • html-gen-utils.scm: functions that ease the production of HTML pages for the null test reports.
  • json-utils.scm: utility functions to handle JSON files produced by the plancknull program
  • user-settings.scm: functions that read user's preferences (currently only from the command line).
